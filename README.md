# reminders_app
A simple reminders app that allows you to create, edit, and manage reminders.

To build and run:
1. Download the `FinalProject.zip` file from the repository.
2. Move the zipped folder to your device's Android Studio project directory (the file path is by default `C:\Users\(username)\AndroidStudioProjects`)
3. Unzip the project folder and open it in Android Studio.
4. You should now be able to edit your own copy of the application.

To install on an Android emulator:
1. Download the `app-release.apk` file
2. Drag and drop over a running Android Studio emulator to install
